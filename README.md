## 3. 프로젝트 관리
- VS Code를 통한 GitHub와의 Clone 연동 후 작업
- GitHub의 프로젝트를 통한 일정, 작업 조율 및 관리
- issue 사용을 통한 예정 사항, 아이디어 회의 및 각자 할 일 정리
- 공동 Repository 생성 후, 개별적인 branch 생성과 팀장을 통한 merge 작업으로 프로젝트 관리
- 일정은 8월 16일부터 시작
- 도움이 될 만한 코드의 예제 실습 후, 참고 사항으로 커밋 예정
